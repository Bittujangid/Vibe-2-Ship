# 🚀 ChronoGuard AI

An AI-powered productivity assistant that helps users plan tasks, optimize schedules, track progress, and meet deadlines using Google's Gemini AI.

## ✨ Features

- 🤖 AI Task Breakdown
- 📅 Smart Schedule Generation
- 📊 Progress Tracking
- ⚠️ Risk Analysis
- ❤️ Health Score
- 🔄 Re-Optimization
- 🚨 Rescue Mode
- 🔐 Firebase Authentication
- ☁️ Cloud Firestore

## 🛠️ Tech Stack

- React + Vite
- Node.js + Express
- Firebase Authentication
- Cloud Firestore
- Google Gemini AI

## 🚀 Getting Started

```bash
# Install dependencies
cd client && npm install
cd ../server && npm install

# Run backend
cd server
npm run dev

# Run frontend
cd ../client
npm run dev
```
